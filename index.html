② パラメータ

💡 各パラメータの役割

Ch2 状態判定閾値 [V]
Ch2 の電圧を見て「充電中 / 待機 / 放電中」の 3 状態に分類する境界です。 |Ch2| > 閾値 なら充電(+) / 放電(-)、それ以下は待機。
通常は 0.1 のままで OK。 Ch2=±0.2V 信号なので 0.1 が最適値です。
ヘッダスキップ行数
CSV 先頭の見出し行数。T&D Graph 標準なら 3 行。
T&D Graph の出力なら 3 のままで OK。
📌 検出ロジック (電圧変化ベース)：各サイクルで
🔵 青 = 放電期間 (Ch2<-閾値) の Ch1 最小値 の点
🔴 赤 = 充電期間 (Ch2>+閾値) の Ch1 最大値 の点
🟡 黄 = Ch2 が 0→-0.2V に切り替わる 1 ステップ前
🟢 緑 = Ch2 が 0→-0.2V に切り替わる 最初の点
🎯 丸の位置(取得箇所)を動かしたい場合
青/赤 を左右に動かしたい 最小/最大値の点で自動決定されるため、パラメータでは動かせません。 検証グラフで [✏️ 編集モード] を ON にして縦線をドラッグしてください。
黄 を動かしたい 緑の縦線をドラッグ → 黄は連動して緑の 1 ステップ前に移動します。
緑 を動かしたい 緑の縦線をドラッグ → その位置が新しい放電開始点になります。
閾値で調整 閾値を小さく→充放電期間が広く認識 ／ 大きく→狭く認識

🖱️ 操作方法
・閲覧モード: マウスホイールで拡大 / ドラッグで範囲選択拡大 / Shift+ドラッグでパン
・✏️ 編集モード ON にすると、縦線に近い位置でドラッグして動かせます
・縦線の±15ピクセル以内にカーソルを近づけると ↔ カーソルに変わります
・ドラッグ可能な縦線: 🔵青 / 🔴赤 / 🟢緑（黄は緑に連動して自動移動）
・緑を動かすと黄も連動（緑の1つ前の時刻に自動設定）
・調整した値は結果テーブル／CSV／Excel に自動反映されます
